Untuk Menjalankan Tailwind buka vscode lalu open folder kelompok kami dan open terminal, berikut perintah yang dapat kalian jalankan pada terminal masing masing

Perintah :
npx tailwindcss -i ./src/assets/css/input.css -o ./src/assets/css/output.css --watch

Setelah menjalankan perintah tailwind di atas, lalu buka file home di folder /src/pages sebagai file Utama dan klik kanan lalu klik "Open With Live Server"
